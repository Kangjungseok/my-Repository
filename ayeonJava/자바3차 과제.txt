가위바위보V3과제

1.기존 메뉴에서 회원가입 기능을 추가하세요.

즉 게임 시작하면 다음처럼 나옵니다.

Welcome Java Game

1.회원가입하기 2.로그인하기 3.현재랭킹 순위보기(승률별) 4.게임 제일 많이 한 랭킹보기 5.끝내기

2.위에서 회원가입을 선택하면, 회원가입 창 또는 콘솔을 띄우고, 이메일 입력 받으세요(당연히 검증로직은 기존과 같으며, 기존 같은 이메일이 있는지 검증하세요. 만약 존재하면 존재한다고 메세지 및 재입력 시키세요)

 이메일 ID 검증이 OK 되면, 암호도 받으세요…암호는 길이검사만 하세요(8~12자 사이) 

모두 OK되면, 축 회원가입 시키는데, 회원가입한 사용자는 D:\ 하위에 thePlayer 라는 폴더 하위에   각각의 사용자 별로 폴더를 생성 후 그 내부에 파일을 생성해서 값을 모두 저장시키세요.

파일명은 id명으로 하던, 메일명으로 하던, 편한대로 구현하세요

가산점 1 > 만약 저장되는 회원 정보를 암호화(AES256) 해서 저장하시면 가산점 10점 더 드림

당연히 불러 올 때도 Decrypt(암호화 해제) 해야합니다. —> 이건 옵션입니다

메뉴 다시 띄우는데, 로그인을 원하면, 로그인 창을 띄우고

id와 암호 입력 받으세요. 입력된 정보가 모두 맞다면, 로그인 되었다고 출력시키고

가장 최근에 접속한 history를 년월시분초로 출력시키세요.

ex> 로그인 됨. 마지막 접속 시간 : 23년 6월 27일 오후 3:24분

로그인 후, 메뉴는 1.게임하기 2.승률별 랭킹보기(10위까지), 3. 게임 수 랭킹보기(10위까지) 4.끝내기

각 메뉴에 따른 결과 보여 주시는데, 다음처럼 하세요.

1. 게임은 한 게임이 끝날 때마다 전체 전적에 누적이 되야합니다. 또는 게임 종료 때 누적해도 됩니다.
2. 단 어떤 경우던 게임 전적은 모두 누적되어야 합니다

ex> 총 게임 300, win 200 loss 100 winingRate 66.66(소수 2자리까지 표현)

모든 사용자는 위처럼 모든 전적을 갖고 있기에, 위에서 랭킹 보기를 하면 원하는 항목대로 보여줘야합니다.

참고로, 메뉴를 추가해서 내림차순, 올림차순으로 보여주기까지 하면 가산점 5점 더 드립니다.

마지막으로 회원탈퇴 기능을 추가하시는데, 이걸 원하면, 반드시 로그인 상태여야 하고, 암호를 

한 번 더 입력 받아 확인합니다.

만약 확인이 되면, 해당 사용자의 모든 파일(폴더 포함) 을 삭제시키고, 회원탈퇴 완료라고 출력시키고 끝내시면 됩니다.
